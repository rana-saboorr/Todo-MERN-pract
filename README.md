# Todo Application

A full-stack Todo application with separate Backend and Frontend components.

## Project Structure

```
Pract/
├── Todo/
│   ├── Backend/     # Backend API server
│   ├── Frontend/    # Frontend client application
└── README.md        # This file
```

## Components

### Backend
The backend server handles API requests and business logic. Navigate to the `Todo/Backend/` directory for setup and development instructions.

### Frontend
The frontend client application provides the user interface. Navigate to the `Todo/Frontend/` directory for setup and development instructions.

## Getting Started

1. **Clone or navigate to the project directory**
   ```bash
   cd Pract
   ```

2. **Set up the Backend**
   ```bash
   cd Todo/Backend
   # Follow the Backend README for installation and setup
   ```

3. **Set up the Frontend**
   ```bash
   cd Todo/Frontend
   # Follow the Frontend README for installation and setup
   ```

## Development

Each component (Backend and Frontend) has its own dependencies and development workflow. Refer to the respective README files in each directory for detailed instructions.
